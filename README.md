# 養孩子，也重新養自己

> 心理師的覺察式育兒練習 — 寫書專案工作區

## 📖 關於這本書

**書名**：《養孩子，也重新養自己》  
**副標**：心理師的覺察式育兒練習

這是一本寫給正在育兒中、曾經是「乖孩子」的父母的書。作者以心理師的專業視角和為人父母的親身經歷，探討一個核心命題：**育兒，不只是養孩子，也是重新養育自己的過程。**

全書整合 IFS（內在家庭系統）、多重迷走神經理論（Polyvagal Theory）與基模治療（Schema Therapy）等臨床視角，讓專業理論透過親身故事落地。

> ⚠️ **正式定稿版本**：`book/plan-c/`（方案 C，第九輪編輯修訂版，96,693 字元 / 約 87,000 中文字）  
> `book/02_contents/` 為原版歷史備份，不再更新。

### 核心框架

- **核心主張**：育兒，不只是養孩子，也是重新養育自己。
- **核心教養姿態**：父母真正要練的，不是控制力，而是**安頓力**。
- **核心順序**：**先安頓，再理解，再教養。**
- **核心流程**：**看見 → 命名 → 安頓 → 重寫**
