# 當了父母，才開始重新養育自己

> 心理師的覺察式育兒練習 — 寫書專案工作區

## 📖 關於這本書

**書名**：《養孩子，也重新好好養自己》

這是一本寫給正在育兒中、曾經是「乖孩子」的父母的書。作者以心理師的專業視角和為人父母的親身經歷，探討一個核心命題：**育兒，其實是重新養育自己的過程。**

全書整合 IFS（內在家庭系統）、多重迷走神經理論（Polyvagal Theory）與基模治療（Schema Therapy）等臨床視角，讓專業理論透過親身故事落地。

### 四篇架構

| 篇 | 主題 | 章節 | 核心提問 |
|---|---|---|---|
| 第一篇 | **覺察** | ch01–03 | 為什麼我會對孩子失控？ |
| 第二篇 | **修復** | ch04–06 | 抱抱那個曾經「太乖」的自己 |
| 第三篇 | **實踐** | ch07–09 | 不必急著教，先學會陪伴 |
| 第四篇 | **傳承** | ch10–12 | 給孩子一份不帶傷痕的愛 |

### 章節一覽

- **前言**：為什麼我會寫這本書
- **ch01** 致新手爸媽：你不是不夠好，你是承受太多
- **ch02** 有一種毒叫童年創傷：你的身體從未忘記
- **ch03** 乖孩子的代價：那些「太安靜」的孩子，正在吞下什麼？
- **ch04** 那個罵你的聲音，其實是過去的回音
- **ch05** 羞愧感：愛自己前的最後一道門
- **ch06** 自我慈悲：你不是壞媽媽，你只是累了
- **ch07** 孩子成功不是靠智商，是靠「大腦 CEO」
- **ch08** 再滑手機我就沒收？你搞錯重點了
- **ch09** 大寶的微創傷：手足之間那些沒說出口的話
- **ch10** 天下無不是的父母？當然有
- **ch11** 復原力：受傷了，還能好好長大
- **ch12** 原來我們都在重新養自己一遍
- **後記**

## ⚠️ 注意事項

1. **素材編碼**：請使用 `fb_posts_fixed.json`，勿使用原始匯出檔（編碼問題）
2. **隱私保護**：個案相關內容已脫敏，修改時維持匿名化
3. **修改方式**：優先編修 `book/02_contents/*.md` 章節工作檔，合併版僅作輸出
4. **風格一致性**：參見 `agent.md` 中的寫作風格要求
5. **術語標準**：統一使用「童年情感忽視」「自我慈悲」「Time Out」等標準用語
