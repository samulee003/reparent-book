# 當了父母，才開始重新養育自己

> 心理師的覺察式育兒練習 — 寫書專案工作區

## 📖 關於這本書

**書名**：《當了父母，才開始重新養育自己：心理師的覺察式育兒練習》

這是一本寫給正在育兒中、曾經是「乖孩子」的父母的書。作者以心理師的專業視角和為人父母的親身經歷，探討一個核心命題：**育兒，其實是重新養育自己的過程。**

### 四篇架構

| 篇 | 主題 | 章節 | 核心提問 |
|---|---|---|---|
| 第一篇 | **覺察** | ch01–03 | 為什麼我會對孩子失控？ |
| 第二篇 | **修復** | ch04–06 | 抱抱那個曾經「太乖」的自己 |
| 第三篇 | **實踐** | ch07–09 | 不必急著教，先學會陪伴 |
| 第四篇 | **傳承** | ch10–12 | 給孩子一份不帶傷痕的愛 |

### 章節一覽

- **前言**：寫給正在育兒路上迷路的你
- **ch01** 當孩子按下了你的按鈕
- **ch02** 那些你以為忘記的，身體都記得
- **ch03** 乖孩子的代價
- **ch04** 和內在小孩重逢
- **ch05** 承認受傷，不是軟弱
- **ch06** 原諒自己，是修復的起點
- **ch07** 陪伴比教導更有力量
- **ch08** 情緒不是敵人
- **ch09** 在衝突中練習靠近
- **ch10** 你給的愛，帶不帶傷？
- **ch11** 打破循環，從覺察開始
- **ch12** 給孩子一份不帶傷痕的愛
- **後記**：育兒這條路，我們都在學

## 🛠️ 技術說明

### 素材處理
- 原始 FB 貼文從 `Downloads/this_profile's_activity_across_facebook/posts/` 匯出
- 經過 `fb_posts_fixed.json` 修正編碼（latin-1 → utf-8）
- 素材分類使用 `build_material_map.py` 及 `scan_fb_posts*.py` 系列腳本

### 寫作工具
- 主要使用 WorkBuddy AI 協作平台進行人機共筆
- 安裝了 `book-writer`、`chinese-content-writing-guideline` 等寫作 Skills
- 審稿採多 agent 團隊協作（總編輯、文編等角色分工）

### 字數統計
- 初稿約 30,000 字（中文字約 23,700）
- 目標：出版級定稿 40,000–60,000 字

## ⚠️ 注意事項

1. **素材編碼**：請使用 `fb_posts_fixed.json`，勿使用原始匯出檔（編碼問題）
2. **隱私保護**：個案相關內容已脫敏，修改時維持匿名化
3. **修改方式**：使用 `replace_in_file` 逐步修改，避免整章覆寫導致遺失
4. **風格一致性**：參見 `agent.md` 中的寫作風格要求
