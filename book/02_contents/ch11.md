# 第十一章 復原力：受傷了，還能好好長大

人生不可能不摔倒。差別在於，摔倒之後你是躺在地上哭，還是爬起來繼續走。

這個能力叫「復原力」（Resilience）。它不是天生的性格特質。它是可以被培養的技能。

作為父母，你在孩子身上種下的每一顆種子 —— 包括那些你後悔的時刻 —— 都可以成為培養復原力的土壤。這一章，我想陪你聊聊：什麼是真正的復原力，怎麼幫孩子長出這份力量，以及，你自己的復原力如何成為孩子最好的教科書。

---

## 11.1 復原力不是什麼

先澄清四個常見的誤解。

### 誤解一：復原力是「堅強」

很多人以為復原力就是「被打倒了還能笑著站起來」。不是。

發展心理學家 Ann Masten 曾將復原力稱為「普通的魔法」（ordinary magic）。復原力不是什麼了不起的超能力，不是麻木，也不是「男子漢不流淚」。它從日常陪伴關係中生長，只需要一個孩子被允許脆弱，在跌倒痛苦之後，能經驗被大人陪伴走過黑暗的歷程，從而慢慢找到重新站起來的力量。一個有復原力的孩子，不是不會難過的孩子，而是難過了還願意再試一次的孩子。

我在臨床上見過太多被要求「堅強」的孩子。有一個國中男孩，父親長期告訴他「男生不要哭」。他在學校被同學排擠，回家一句話不說。媽媽問他怎麼了，他只說「沒事」。後來他開始出現胃痛、失眠、上學前會嘔吐。身體替他說了嘴巴不許說的那些話。

復原力從來不是忍住不哭。能哭出來的孩子，其實比憋著的孩子更有力量。

### 誤解二：復原力是「天生的」

有人說「這孩子天生韌性強」「那孩子就是比較脆弱」。這種說法既不科學也不公平。

許多長期追蹤研究發現：即使在高風險家庭中成長，只要孩子生命中擁有至少一個穩定的、支持性的成人關係，他們日後的心理健康與復原力就會顯著提高。這印證了環境與經驗的影響力非常大 —— 作為父母，你不需要完美，你只需要穩定地「在場」。

### 誤解三：復原力是「獨自扛過去」

有些人以為復原力就是「自己消化、不依賴別人」。這其實是孤獨，不是復原力。

真正的復原力需要至少一個「安全的人」—— 一個在你跌倒時願意陪你、傾聽你、讓你知道「你不是一個人」的人。這個人可以是父母、老師、朋友、或者任何一個值得信任的大人。

我服務過一位年輕媽媽，她的女兒在幼稚園被同學孤立，回家後大哭。媽媽的第一反應是：「你別理她們就好了啊，自己玩也可以啊。」女兒哭得更厲害了。後來她換了方式，把女兒抱在懷裡說：「被朋友排擠一定很難過，媽媽在這裡。」女兒的哭聲漸漸小了。那天晚上，女兒自己拿了一張畫紙，畫了三個小人手牽手，說：「這是妳、我和妹妹。」

「自己扛」是一種生存策略，但不是長遠之計。復原力的底層結構是人與人之間的連結。一個知道「有人接住我」的孩子，才敢在跌倒之後重新站起來。

### 誤解四：男孩不需要太多情緒支持

「男生就是要堅強」「男孩子哭什麼哭」—— 這兩句話，在華人社會裡幾乎無所不在。

但心理學研究告訴我們另一個故事。PISA（國際學生能力評量計畫）曾經測量各國學生的「害怕失敗指數」。結果發現，在所有參與國家中，害怕失敗的男孩比例比女孩高。換句話說，男孩對「失敗」的恐懼，其實比女孩更深。

只是男孩不被允許說出來。

他們把恐懼轉化為憤怒、攻擊、或沉默。他們用「我不在乎」掩飾「我很害怕」。他們用行動取代語言 —— 打人、摔東西、逃學 —— 因為沒有人教過他們怎麼用語言表達內心的不安。

我在諮商中見過一個國中男孩，被老師轉介來，原因是「經常跟同學起衝突」。我問他最近有什麼事讓他生氣。他想了很久，說：「沒有。我不會生氣。」

後來他慢慢說出，他覺得家裡所有的大人都偏心弟弟。弟弟可以哭、可以鬧、可以被哄。但他不能。因為他是哥哥。

「哥哥不能哭。」這句話他重複了三次。每一次聲音都更小一點。最後一次，他的眼眶紅了。但他沒有讓眼淚掉下來。因為他已經被訓練得很好了。

如果你的家裡有男孩，請多給他一個許可：你可以難過。你可以害怕。你可以說「我不知道怎麼辦」。這些話不會讓他變弱。會讓他變成一個有能力面對自己真實感受的人。

## 11.2 培養復原力的四個支柱

心理學研究指出，復原力的培養有四個關鍵支柱。它們不是什麼高深的技術，而是嵌入在日常生活中的小選擇。

### 支柱一：至少一個穩定的成人關係

這是所有研究裡一致公認最重要的因素。

孩子不需要十個完美的成人陪伴者。他只需要至少一個穩定的、可靠的、願意在他面前「在場」的成人。

這個人不需要完美。他可以生氣、可以犯錯、可以笨拙。但他需要是「穩定」的 —— 不會因為孩子表現不好就消失、不會因為自己的情緒而變得不可預測。

** 為什麼這麼重要？**

從依附理論的觀點簡而言之：孩子小時候若經歷「我難過時，有人會來接住我」，長大後就會相信世界大致安全、自己值得被愛；反之則會學到「別人不可靠，只能靠自己」。但好消息是，即使早期關係品質不理想，日後遇到穩定的關係（親人家屬、老師、伴侶或心理師），這份信任感依然能被修補。

如果你是你孩子生命中這個穩定的成人，那就夠了。你不需要做到百分百，做到「大部分時候都在」就已經是一份很了不起的禮物。

我有一位同事，她的母親在她小時候經常因為工作忙而不在家。但她的外婆每天都在。外婆不識字，不會跟她講什麼大道理，就是每天下午煮好飯等她放學，坐在她旁邊看她寫作業，偶爾問一句「今天學校好不好玩」。多年後，這位同事成為了心理師，她跟我說：「我其實不記得外婆說過什麼深刻的話，但我記得那個『有人在等我的』感覺。那個感覺，讓我在後來人生中遇到困難的時候，覺得自己不是一個人。」

### 支柱二：情緒詞彙

能夠用語言描述自己的感受，是復原力的基礎。

研究顯示，能夠精確命名情緒的孩子（「我不只是難過，我感到被背叛」），比只能說「不好受」的孩子，更容易從創傷中恢復。神經科學家 Lisa Feldman Barrett 的研究指出，情緒詞彙越豐富的人，大腦的前額葉皮層越能有效地調節杏仁核的活動。白話一點說：能說出「我現在什麼感覺」的孩子，大腦比較不容易被情緒劫持。

怎麼幫孩子建立情緒詞彙？很簡單：幫他命名。

當他哭的時候，你不是說「不要哭」，而是說「你現在是不是覺得很委屈？因為同學沒有邀請你。」

當他生氣的時候，你不是說「生氣也沒用」，而是說「你很生氣對不對？因為你覺得不公平。」

每一個情緒被命名的瞬間，都是大腦前額葉在跟杏仁核說話的過程。練得越多，孩子就越能自我調節。

** 不同年齡層的命名示範：**

幼兒（2-5 歲）只需要簡單的詞彙：「你很開心」「你很生氣」「你好像有點害怕」。這個階段的重點不是精確，而是讓孩子知道「感受有名字」。

學齡期（6-12 歲）可以開始引入更細緻的詞彙：「你覺得這件事不公平，所以很氣憤」「你跟好朋友吵架了，是不是既難過又懊惱？」你可以用「是不是覺得……」的句型，幫孩子猜，而不是直接告訴他。

青少年（13 歲以上）可以嘗試更複雜的情緒語言：「你現在是不是有點矛盾？一方面想獨立，一方面又覺得需要幫忙」「那件事讓你覺得被誤解了，對吧？」青少年最怕被「教育」，你可以多用「我觀察到」「我猜」的開頭，留空間讓他自己接話。

** 給讀者的情緒詞彙參考表：**

| 基本情緒 | 進階情緒 | 更精細的描述 |
|---------|---------|------------|
| 難過 | 委屈、遺憾、失落 | 被忽略的寂寞、沒被看見的空虛 |
| 生氣 | 氣憤、煩躁、不甘 | 被誤解的委屈、努力被否定的挫敗 |
| 害怕 | 焦慮、緊張、不安 | 對未知的恐懼、怕讓人失望的壓力 |
| 開心 | 滿足、感恩、驕傲 | 被認可的踏實、努力的成果被看到的喜悅 |
| 尷尬 | 羞愧、丟臉、難為情 | 被當眾指出錯誤的無地自容 |

這份表不是讓你拿來考孩子的，而是給你自己參考 —— 當你不知道怎麼幫孩子命名時，可以從中找到適合的詞。

### 支柱三：掌控感的建立

創傷的核心體驗之一是「無力感」—— 我對發生在我身上的事情沒有控制權。

培養復原力的方法是：在安全的範圍內，給孩子越來越多的選擇權和掌控權。

三歲的小宇早上不肯穿外套，媽媽說：「今天冷，不穿外套會著涼，一定要穿。」小宇躺在地上尖叫。第二天媽媽換了策略：「今天有兩件外套，你要紅色的還是藍色的？」小宇想了想，說「藍色的」，然後自己穿上了。

這不是什麼高深的技術，只是一個小小的選擇權。但對三歲的孩子來說，「我自己決定的」跟「被迫的」是完全不同的體驗。

小學二年級的美美不喜歡上鋼琴課，每次去之前都哭。媽媽跟她聊了很久，發現她不是不喜歡音樂，是每次都被規定要練指定的曲子，讓她覺得「這不是我的」。後來媽媽跟老師商量，每週讓美美自己選一首她想彈的歌。幾個月後，美美開始主動練琴了。

十四歲的阿傑想參加學校的籃球隊，但練習時間跟補習衝突。爸爸沒有直接替他決定，而是跟他一起列出「參加的好處」「不參加的好處」「可能遇到的困難」，讓他自己做決定。阿傑最後選擇了參加籃球隊，他說：「雖然可能會累，但這是我的選擇。」

每一個「我自己決定」的瞬間，都在告訴孩子：你有能力影響你的生活。這種感覺累積起來，就是面對逆境時「我不會完全無助」的底氣。

### 支柱四：失敗的日常化

如果一個孩子從來沒有失敗過，他怎麼學會應對失敗？

讓孩子在安全的環境裡體驗失敗。比賽沒得名次、考試考砸了、跟朋友吵架了、被拒絕了。這些都是練習。

你的角色不是幫他避免失敗，而是在他失敗之後陪他走過去。問他：「發生了什麼事？你現在感覺怎麼樣？你覺得下次可以怎麼做？」

讓他從你的態度裡學到：失敗不是世界末日。它只是生活的一部分。

但說起來容易，做起來很難。很多父母在孩子失敗的時候，自己的情緒比孩子還激動 —— 比賽輸了，爸爸比孩子還生氣；考試考差了，媽媽比孩子還焦慮。這些反應背後，往往是我們自己小時候的經驗在作祟。我們那個年代，「失敗」代表被罵、被嘲笑、被認為「不夠好」。所以我們長大後，看到孩子失敗，身體裡那個小時候被罵的自己就醒了過來。

如果你發現自己有這個模式，可以先做一件事：在孩子失敗的時候，先安撫自己。深呼吸，告訴自己：「這是他的經驗，不是我的。他需要的是一個陪他的人，不是另一個壓力源。」

心理學有一個概念叫「創傷後成長」（Post-Traumatic Growth, PTG）。研究者 Tedeschi 和 Calhoun 發現，有大約 50% 到 70% 經歷過重大創傷的人，不僅能恢復到創傷前的狀態，甚至在某些面向上變得「比以前更好」—— 他們對人際關係的珍惜更深了，對生命的意義感更強了，對自己的能力也更有信心了。

這不是說創傷是好事。創傷很痛，沒有人應該被創傷。但人類的身心系統有一種驚人的修復能力。關鍵在於：這個人是不是有一個安全的環境去消化這段經歷、是不是有人陪他一起找到意義、是不是被允許慢慢來。

對孩子來說，日常的小失敗就是創傷後成長的「預防針」。每一次他從失敗中站起來，都在告訴自己：「我犯錯了，但我沒有碎掉。」

## 11.3 父母的復原力，是孩子的模板

最後，我想提一個常被忽略的事實：孩子的復原力，很大程度上是在模仿你的。

如果你在孩子面前摔倒之後能夠坦然地說「哎呀摔倒了，好痛，沒關係再試一次」，你的孩子就會學到：摔倒不是末日。

如果你在孩子面前遇到挫折時總是暴怒、自責、崩潰，你的孩子就會學到：失敗是可怕的。

你不是要當一個「不會失敗」的父母。你是要在失敗的時候示範：怎麼接受它、怎麼處理它、怎麼走過去。

我想分享自己的經歷。

幾年前，我花了好幾個月準備一場大型專業演講。從題目設計、文獻整理、投影片製作到講稿撰寫，前前後後投入了至少一百個小時。演講當天，我興奮又緊張地站在台上，開場沒多久，投影設備突然故障，螢幕一片黑。台下幾百雙眼睛看著我。

那個瞬間，我的大腦一片空白。心跳加速，手心冒汗，臉部發燙。我可以感覺到恐慌像潮水一樣湧上來。

我深呼吸了一下，對著觀眾說：「看來投影有點想法，那我們就先用說的吧。」台下有人笑了。我放下講稿，用最自然的方式把內容講完。結束的時候，有好幾位參與者走過來說：「妳處理得好好，一點都看不出來妳緊張了。」

回家那天晚上，孩子看到我，問：「媽媽今天好玩嗎？」我蹲下來跟她說：「今天發生了一點小狀況，投影機壞了，媽媽本來有點慌張，但後來還是講完了。」

女兒歪著頭想了想，說：「所以有時候事情會不聽話，但我們還是可以做完？」

「對，就是這樣。」

那一次，我做的其實就是本書前面提到的幾件事：我沒有壓抑自己的緊張（情緒詞彙 —— 我說出了「有點慌張」），我承認了自己的脆弱，然後我帶著脆弱繼續完成了該做的事。

我不知道這段對話在她心裡留下了什麼。但我知道，那天晚上我在做的，就是在示範「怎麼在事情不順利的時候，不讓自己崩潰，而是帶著慌張繼續往前走」。

還有一次，我不小心把女兒手做了一整個下午的手工撕壞了。她當場大哭。我的第一反應是自責 ——「你怎麼這麼粗心」「明明是她最珍惜的東西」。那個自我批評的聲音差點就要衝出來。但我忍住了，蹲下來看著她，說：「媽媽真的很抱歉，是媽媽不小心弄壞的。媽媽知道妳很傷心，因為那是妳很用心做的。」女兒哭了很久，後來她說：「媽媽，你弄壞了但是你沒有騙我說是它自己掉的。」

我不知道她哪來的這種觀察力。但那句話提醒了我：孩子在看的不只是你做了什麼，還有你怎麼面對你做錯的事。道歉、承認錯誤、承擔後果 —— 這些都是復原力的具體表現。當父母願意示範這些，孩子就會學到：犯錯不可怕，可怕的是不承認、不面對。

**「你的孩子不記得你對他說過的每一句道理，但他會記得你面對困難時的樣子。」**

## 11.4 兒童復原力量表：你的孩子夠「有彈性」嗎？

以下是一個簡單的自我評估工具，幫你觀察孩子的復原力狀態。這不是診斷工具，只是一個讓你更有方向去了解孩子的參考。

請回想孩子最近六個月來的表現，在每個項目上勾選「經常」「有時」或「很少」。

| 項目 | 經常 | 有時 | 很少 |
|------|------|------|------|
| 1. 遇到困難時，願意嘗試不同的解決方式 | | | |
| 2. 失敗或被拒絕後，能在合理的時間內恢復情緒 | | | |
| 3. 能用語言表達自己的感受（不只是哭或發脾氣） | | | |
| 4. 願意尋求幫助（向父母、老師、朋友） | | | |
| 5. 面對新環境或新挑戰時，雖然緊張但願意嘗試 | | | |
| 6. 能從失敗的經驗中學到東西（「下次我會……」） | | | |
| 7. 有一個他信任、願意分享心事的人 | | | |
| 8. 在日常生活中有自己能做主的範圍（選衣服、選活動等） | | | |
| 9. 遇到不順利的事，不會立刻認定「我就是不行」 | | | |
| 10. 跟朋友吵架後，有修復關係的意願和行動 | | | |

** 怎麼看結果：**

「經常」6 個以上：孩子的復原力發展得不錯。繼續保持你正在做的事。

「經常」3-5 個：有基礎，但某些面向可以加強。可以對照前面的四個支柱，看看哪一塊比較弱。

「經常」2 個以下：孩子可能需要更多的支持。這不代表他「有問題」，而是他還在等一個更安全的環境、一個更穩定的陪伴者。如果孩子在情緒上長期處於困擾狀態，建議尋求專業心理師的協助。

請記住：這份量表觀察的是「目前的狀態」，不是「永遠的標籤」。孩子的復原力會隨著環境和經驗而改變。而你能做的，就是提供那個讓他改變的環境。

## 一個從挫折中站起來的故事

我想用一個完整的故事，把這一章的內容串起來。

小傑是個五年級的男孩，他從小就喜歡畫畫。三年級的時候，美術老師在全班面前表揚他的作品，他高興了好幾天。從那時候開始，他一直覺得自己是班上「畫得最好的人」。

五年級的學校美術比賽，他信心滿滿地報了名。準備了兩個星期，每天放學後都留在教室畫。比賽那天，他帶著自己最滿意的作品去交件。結果公布，他沒有得獎。連優選都沒有。

回家的路上，他一句話都沒說。媽媽從他的表情裡看出來了，問他：「比賽結果怎麼樣？」

「沒得名次。」他的聲音很輕。

媽媽忍住「沒關係啦下次再努力」的衝動，說了一句：「那一定很難受吧。」

小傑突然就哭了。他說：「我準備了那麼久，結果什麼都沒有。我一定很爛。」

媽媽沒有急著安慰他，而是坐在他旁邊，安靜地等他哭完。然後問他：「妳願意跟媽媽說說，最讓妳難過的是什麼嗎？」

「就是…… 我以為我畫得很好，結果別人畫得比我好太多。我以前覺得自己很厲害，現在我覺得自己什麼都不是。」

「嗯，妳不是『什麼都不是』。妳是發現了，原來有比自己更厲害的人。這個感覺的確很衝擊。」

那天晚上，媽媽沒有多說什麼。她只是做了一件事 —— 把小杰以前畫的畫，從冰箱上、從房間牆上，一張一張取下來，跟他一起看。「你看，這幅是小班時候畫的，這幅是一年級的，這幅是去年的。你看，妳的畫一直在進步，對不對？」

小杰看著那些畫，表情慢慢從沮喪變成若有所思。「好像…… 是有變好一點。」

「比賽沒得名次，不代表妳沒有進步。它只代表，在那一天、那些評審的眼裡，有別人的作品更符合標準。但妳的畫，對媽媽來說，一直都是最好的。」

幾天後，小杰跟同學聊天時提起了這件事。同學說：「我上次參加作文比賽也沒得名次，我後來才知道，全班只有五個人報名，最後得獎的都是六年級的。」小杰笑了：「那我也沒輸給五年級的啊。」他有點驚訝地發現，原來「沒得名次」不代表「我很差」，也可能只是「評審的標準跟我不一樣」。

媽媽後來跟我分享這件事時說了一個讓我很感動的細節。那天晚上，小杰睡覺前突然跟媽媽說：「謝謝妳沒有說『沒關係啦』。」媽媽愣了一下，問：「為什麼？」

「因為『沒關係』就是說這件事很嚴重，妳在安慰我。但我當時不是想被安慰，我只是想有人知道我很難過。」

一個十一歲的孩子，能說出這樣的話。這就是情緒詞彙和穩定關係一起作用的結果。他不但能命名自己的感受，還能分辨「被安慰」和「被理解」的不同。

第二天，小傑主動拿出畫簿開始畫新的東西。媽媽偷偷瞄了一眼，畫的是一個站在頒獎台旁邊的小人，手裡拿著自己的畫，嘴角有點往下，但背挺得很直。旁邊寫了一行字：「下次，我會再試一次。」

## 補記：當枕邊人「衝康」你的一百種日常

寫了這麼多嚴肅的內容之後，我想用一個輕鬆的故事結束這一章。

因為真實的婚姻生活，不只有深度的情感交流。還有那些讓你哭笑不得的日常。

### 如坐針氈的母親節

夫妻都是臨床心理師，可能會讓外界有許多夢幻的想像：無數個深夜的促膝長談、各式心靈雞湯的靈性對談、一眼能夠看穿對方拗執背後的真實渴望……

X啦。

不知道其他心理師夫妻是如何攜手共度相愛相殺的日子，至少我們這幾年的夫妻生活（對我而言）絕對是無數個白眼與血淚辛酸交織堆砌而成。

母親節那天，與先生家人相約一同聚餐，慶祝母親節與先生的生日。公婆一到餐廳就拿出上面寫滿祝福語句的紅包給寶貝兒子。

先生欣喜地拿了紅包立刻轉身大聲宣告：「老婆！這紅包給妳，母親節快樂！」

聽到這句話，我差點沒把嘴裡剛喝的水噴在他臉上。眼角瞥見先生身後公婆變臉的尷尬與驚訝表情——

見此宛如六月雪般瞬間凝結冷凍的空氣，我都快要顧不得八個多月的身孕，直接趴跪在地、高舉申冤令牌、擊鼓鳴冤：

**「大人，冤枉啊啊啊啊！媳婦我從沒從您們兒子手中拿過一分一毫您們贈予他的紅包啊！」**

但我知道一切多說無益。黑了就是黑了。再多解釋、再瓊瑤式的哭喊都沒路用了。畢竟他們倆老不是包青天，只是一對愛子心切、深怕兒子受媳婦荼毒而缺衣少食的憂心父母。

而我，就是那個每天讓老公過付清節、連生日紅包都不放過的黑心惡媳婦。

話說先生說完那句話之後很順手將紅包塞回自己褲袋。既然已經黑了是不是就順勢叫他交出來？嗯……算了。

**黑心惡媳晚餐還很認命的掏錢付帳。枕邊人段數太高不得不慎防。**

分享這個故事是因為：很多父母讀到「跟伴侶好好溝通」這類建議時，會覺得壓力很大，好像每一次對話都要正經八百、有理有據。但其實不是這樣的。真實的婚姻生活裡，更多時候是互相吐槽、一起翻白眼、然後在笑聲中翻篇。

能一起笑著面對彼此的不完美，本身就是一種很棒的復原力。
