## 全書架構評估（8/10 分）

**優點：**
- 四篇章遞進結構清晰：覺察→修復→實踐→傳承，符合「先看見問題→再處理自己→然後面對孩子→最後回到家庭系統」的邏輯
- 每篇三章的安排大體合理，各篇內部有從「認識問題」到「具體方法」的遞進
- 前言有效引導讀者進入全書，用小薇的故事破題，四篇章概述清晰
- 後記回扣前言小薇的故事，形成首尾呼應

**扣分原因：**
- 第九章「身體界線」（9.8）與手足競爭主題脫節，且該主題更適合放在第三篇「實踐」或第十章「傳承」
- 第十一章 11.1「誤解四」標號跳躍（誤解一→誤解四→誤解二→誤解三），明顯是遺珠素材插入後未重新編號
- 第三篇「實踐」偏重學齡前至小學的場景，對青少年的著墨較少，與前言所述「3C衝突的關鍵不在時間」的承諾有落差

## 各章審查

### 前言
- 「前言概述了四篇章，但第二篇說『你會學到什麼是童年情感忽視』——然而 CEN 的正式介紹其實在第三章 3.4，第四章只是回顧式提及。前言的描述可能讓讀者期待第二篇才詳細講 CEN，造成閱讀預期錯位。 — P1 — 建議修改為：第二篇描述改為「你會深入認識那個罵你的聲音、理解羞愧感如何影響你，並開始練習自我慈悲」，將 CEN 的學習放在第一篇的範圍內更準確。

### 第一章 致新手爸媽：你不是不夠好，你是承受太多
- 1.6「為什麼育兒特別容易觸發舊傷」與第二章 2.6 標題完全相同，且兩節探討的核心問題高度重疊。第一章 1.6 的功能是「預告」，第二章 2.6 是「深入展開」，但讀者連續閱讀時會產生重複感。 — P1 — 建議：第一章 1.6 精簡為一個簡短的過渡段（3-5行），點出「舊傷」概念即可，把完整的四個觸發場景留到第二章。同時修改 1.6 標題避免與 2.6 完全相同。
- 第一章使用化名「小薇」（前言中的案例），1.6 又出現另一位媽媽、一位爸爸、以及作者自己的案例，案例密度偏高，節奏略顯擁擠。 — P2 — 可考慮將 1.6 中的一個案例移至第二章作為開場。

### 第二章 有一種毒叫童年創傷：你的身體從未忘記
- 2.1 小雯的故事中，她「抓住女兒的肩膀，吼了一句」，但前言小薇的故事也是吼孩子、也是「發誓不要變成我媽」。兩個案例的敘事結構太相似（都是母親→發誓→失控→發現跟媽媽一樣），讀者可能混淆。 — P1 — 建議：在小雯故事開頭加一句區隔（如「跟小薇不同，小雯的童年不是暴力，而是不確定」），或改變小雯案例的核心特徵（如聚焦在她「逃避」而非「吼回去」的模式）。
- 2.2 ACEs 段落中，「subtler」一詞直接使用英文，全書其餘地方未出現此用法。 — P2 — 改為中文「更隱微的」。
- 2.3 三種逆境類型（剝奪、威脅、不可預測）的案例人物（小志、曉雯、美琪）與 2.1 的小雯、2.4 的小敏名字容易混淆（小志/小雯/小敏/曉雯/美琪），短距離內出現太多化名。 — P2 — 建議統整化名，避免同音或相似字。

### 第三章 乖孩子的代價：那些「太安靜」的孩子，正在吞下什麼？
- 3.4 CEN 正式介紹的位置合理（第三章屬覺察篇），但第四章 4.3 又用一段回顧 CEN，第五章 5.2 再提及「被忽略也是一種羞愧」。三處提及 CEN 的深淺度不一，第四章的回顧較為冗餘。 — P1 — 建議：第四章 4.3 對 CEN 的回顧精簡為一句話帶過（如「還記得第三章談過的童年情感忽視嗎？」），不要重新解釋定義。
- 3.5 CEN 自我辨識清單 10 題，第 9 題出現「亲密」一詞（簡體字），全書其餘均為繁體中文。 — P0 — 必須修正為「親密」。
- 3.7 修復練習（練習一二三）與第三章其他部分相比篇幅偏短，且「練習三：找到一個安全的傾聽對象」與第六章 6.4 及第五章 5.4 的內容高度重複。 — P2 — 可考慮在此處只點到為止，引導讀者到後面章節看詳細說明。

### 第四章 那個罵你的聲音，其實是過去的回音
- 4.5 IFS 內在家庭系統段落中，虎媽的故事與第一章 1.6 作者提到「嚴格高要求的虎媽」重複。第一章已說過「女兒曾經說：『媽媽每次溫習的時候，好像變了一個人。』」，第四章 4.5 又重複了幾乎相同的對話。 — P0 — 這是第一輪去重後仍殘留的重複。建議：第四章 4.5 在提到虎媽時，用「還記得第一章提過的虎媽嗎？」的方式回溯，不要重述整個故事和對話。
- 4.4 步驟五「寫一封信給小時候的自己」與第十二章 12.6 第二週練習完全重複。 — P1 — 建議：第四章的版本保留為「初次介紹」，第十二章改為「還記得第四章的寫信練習嗎？這次，試著再寫一封，看看跟上次有什麼不同」，避免讀者覺得在重複做同一件事。

### 第五章 羞愧感：愛自己前的最後一道門
- 5.3 面具人格的三個案例（便當媽媽、不抱怨爸爸、回娘家媽媽）品質良好，但「回娘家裝幸福的媽媽」案例與第一章 1.1 中「不被允許喊累」的主題重疊。 — P2 — 可接受，因為第五章聚焦的是「羞愧感」驅動的面具，第一章是「社會期待」，角度不同。但若要精簡，可合併。
- 5.4 練習三「找到你的反羞愧人」中的替代方案（專業協助、線上社群、寫日記、對自己說），與第六章自我慈悲練習中「練習時可能遇到的阻力」段落有功能重疊。 — P2 — 兩處各自獨立可讀，但全書統讀時略有冗餘感。

### 第六章 自我慈悲：你不是壞媽媽，你只是累了
- 6.1 第三段和第四段完全重複：「如果連專業的助人工作者都會在面對他人脆弱時急著「修正」，你對孩子做同樣的事，一點都不奇怪。因為你也從來沒有被好好「同在」過。」出現了兩次（第 52-53 行）。 — P0 — 必須刪除其中一段。這是明顯的編輯遺留。
- 6.2 自我善待段落中，「研究給了我們答案。2021 年《Journal of Traumatic Stress》的一項研究指出……」與 6.5 迷思一中「在一項針對戰鬥軍人的研究中，缺乏自我慈悲比戰鬥暴露的程度更能預測 PTSD 症狀的嚴重性」指的可能是同一項研究，但一處有出處、一處無出處。 — P1 — 建議統一：若為同一研究，在 6.5 處補充出處或標註「前面提過的研究」；若為不同研究，需釐清。
- 6.2 自我善待段落末尾的「請對自己說……」轉折較突兀，從研究分析直接跳到自我對話，缺少過渡。 — P2 — 可加一句過渡：「所以，下一次那個聲音又響起來的時候，試著這樣對自己說——」

### 第七章 孩子成功不是靠智商，是靠「大腦 CEO」
- 7.3 倒數計時段落中提到「還記得第八章提到的那個故事嗎？」（第 121 行），這是向後引用（第七章引用第八章），在書籍排版中會造成讀者困惑。 — P0 — 向後引用在書籍中不常見且不直觀。建議：刪除此句，改為「還記得前面提過的故事嗎？」或直接在此處簡述，不指向後面的章節。若此故事確實在第八章有更完整的版本，可在第八章再完整呈現。
- 7.5 內在動機段落中，大寶 iPad 的故事與第八章 8.3 完全重複（長針指到 6、追劇忘記時間、大寶準時關掉）。 — P0 — 同一故事在兩章中以幾乎相同的文字出現。建議：第七章保留此故事但精簡為 3-4 行概述（聚焦在「內在動機」角度），第八章再完整展開（聚焦在「3C 衝突中的自主感」角度）。
- 7.6 在家訓練 EF 的「從人的心也在的陪伴開始」段落，與 7.5 末尾的「人在心也在的陪伴」內容幾乎逐字重複。 — P1 — 建議合併或刪減其中一處。
- 第七章涵蓋主題過多：執行功能 + 內在動機 + 讚美藝術 + ADHD，章節篇幅明顯超出其他章節。 — P1 — 建議：考慮將「讚美的藝術」（7.7）移至第十一章復原力中作為「培養成長型心態」的子主題，或將 ADHD（7.8）獨立為專欄。目前章節長度失衡。

### 第八章 再滑手機我就沒收？你搞錯重點了
- 8.3 自主感段落中的大寶 iPad 故事與第七章完全重複（見上方 P0 條目）。
- 8.6 方向三「豐富螢幕以外的選項」最後一段（第 231 行起）又出現了大寶願意準時關平板的原因描述，與 7.5 和 8.3 的內容第三次重複。 — P0 — 同一故事在全書出現三次。必須刪減至只在一處完整呈現，其餘用簡短回溯。
- 8.1 作者提到自己對螢幕的焦慮（大寶幼稚園要求平板），與 7.5 開頭的描述基本相同但用語略有差異。 — P2 — 建議統一表述。

### 第九章 大寶的微創傷：手足之間那些沒說出口的話
- 9.2 出現了兩個 9.2（「分離，也可以是一種創傷」和「大寶的行為，不是『變壞了』」）。第一個 9.2 討論分離創傷，第二個 9.2 討論行為變化。 — P0 — 必須重新編號。建議：「分離創傷」改為 9.2，「大寶的行為」改為 9.3，後續章節依次調整（9.3→9.4, 9.4→9.5, 9.5→9.6, 9.6→9.7, 9.7→9.8, 9.8→9.9, 9.9→9.10）。
- 9.8「身體界線」主題與手足競爭的核心主題關聯薄弱。身體界線更多是「保護孩子」的通用議題，不是手足特有的問題。 — P1 — 建議：若保留此段落，需加強與手足主題的連結（例如：手足之間的身體界線——哥哥可以隨便碰弟弟嗎？），或將此主題移至第十章「傳承」篇中作為「跨世代教養」的子主題。
- 9.7「男孩的悄悄受傷」與第十一章 11.1「誤解四：男孩不需要太多情緒支持」高度重疊，連案例（國中男孩、偏心弟弟、「哥哥不能哭」重複三次）都幾乎一樣。 — P0 — 同一案例在兩章重複出現。建議：第九章保留男孩主題但更換案例（聚焦在手足情境中的男孩），第十一章使用現在的案例（聚焦在復原力誤解）。

### 第十章 天下無不是的父母？當然有
- 10.2 作者提到「長輩聽聞寶寶沒有每天排便，便驚呼……」與第一章 1.2 完全相同的事件。 — P1 — 第一章已用此案例說明「關係品質下降」，第十章再用來說明「教養衝突」角度不同，可接受。但建議在第十章加一句回溯：「還記得第一章提過的那個母奶事件嗎？」讓讀者知道這是同一件事的不同角度。
- 10.3 場景三（母奶排便問題）與 10.2 的案例又是同一件事的第三度出現。 — P1 — 建議 10.3 場景三改用另一個長輩質疑的例子（如副食品添加時機、哭要不要抱等），避免同一事件出現三次。
- 10.7 底線二中突然插入「過年聚會」段落（第 300 行），與「界線=柵欄」的比喻邏輯斷裂。 — P1 — 過年聚會的主題在 10.3 場景七已有詳細處理，此處重複且打斷了底線論述的連貫性。建議刪除此段，或移至 10.3 場景七後作為補充。

### 第十一章 復原力：受傷了，還能好好長大
- 11.1 標號錯誤：出現「誤解一」「誤解四」「誤解二」「誤解三」的順序。 — P0 — 必須修正為「誤解一、二、三、四」的正確順序。「誤解四：男孩不需要太多情緒支持」應排在最後。
- 11.1「誤解四：男孩不需要太多情緒支持」與第九章 9.7 高度重疊（見上方 P0 條目）。
- 11.2 支柱一「至少一個穩定的成人關係」中，依附理論的介紹（Bowlby、Ainsworth）篇幅偏長，且與第二章 2.5「大腦會修復」中關於安全關係的論述有重疊。 — P2 — 可考慮精簡依附理論的學術背景，聚焦在「對父母的具體意義」。

### 第十二章 原來我們都在重新養自己一遍
- 12.1 引用 Gabor Maté 的話（「要讓創傷發生，你只需要……」）與第二章 2.3 已引用的完全相同。 — P1 — 同一句引言在全書出現兩次。建議第十二章改為：「正如第二章提過的那句話——」，不重複引述。或者，第十二章改用 Maté 的另一段話。
- 12.5 再次引用 Gabor Maté 同一段話的第三度出現（第 384 行）。 — P0 — 同一句話在全書出現三次（ch2、ch12 兩處），必須刪減至只在一處完整呈現。
- 12.3「重新養自己的五個日常」中的「每天對自己做一件溫柔的事」與第六章 6.3「每日練習：給自己的三分鐘」功能重疊。 — P2 — 可接受，因角度不同（第六章是具體練習步驟，第十二章是日常心態），但建議在第十二章加一句交叉引用。
- 12.3「問自己：這是我的需求，還是孩子的需求？」是一個極有價值的覺察練習，但在前面的章節中缺乏鋪墊，突然出現略顯突兀。 — P2 — 可考慮在第四章或第五章的自我批評/羞愧感段落中提前埋下這個問題。
