# 第九章 復原力：在受傷之後長出力量

很多父母聽到「復原力」這三個字，第一反應是：那我要把孩子教得更能撐、更不怕挫折、更有抗壓性。

這個方向沒有錯，但如果只理解成「變堅強」，很容易走偏。

因為真正的復原力，不是把孩子訓練得不會痛，而是讓他在痛的時候，不會完全失去自己。

他可以受傷。

可以難過。

可以失敗。

甚至可以在一段時間裡狀態很差。

但他知道怎麼求助、怎麼回來、怎麼在跌倒之後慢慢把自己收回來。

這種能力，不是靠說教長出來的。它長在關係裡，也長在孩子日常怎麼理解自己。

上一章談的是父母怎麼「重寫」自己的回應模式。這一章把鏡頭轉回孩子：當父母開始安頓自己、不再只是重演舊傷，孩子身上會長出什麼？答案就是復原力。

這也意味著，這一章其實還是在說「重新養育自己」。父母願意面對自己的脆弱、學著在跌倒後站回來，孩子才能從中學到：摔倒不是末日，人可以不完美，卻仍然有力量。你的復原力，就是孩子最早看見的那個榜樣。

---

## 9.1 復原力不是硬撐，而是能在痛裡維持連結

有些大人很會誤認復原力。

看到一個孩子哭得少、話不多、很能忍，就說這孩子很堅強。

看到一個孩子受委屈後還若無其事，就說這孩子很有韌性。

可很多時候，那不叫復原力，那叫關閉。

關閉和穩定看起來有時很像。尤其在重視懂事、重視不麻煩別人的文化裡，更容易被誤認。

真正有復原力的孩子，不是沒有反應，而是有反應之後還能慢慢回來。他會知道自己很難過，也知道那份難過不是世界末日。他可能哭、可能氣、可能需要時間，但他沒有因此完全斷線。

這裡的「斷線」有很多種樣子：

- 對自己斷線，感覺不到需要
- 對他人斷線，不相信有人會接住
- 對未來斷線，覺得怎樣都沒用

青少年憂鬱時，常常就會讓人看見這種斷線。很多家長一開始以為孩子只是懶、只是愛頂嘴、只是作息亂掉，後來才發現那不是單純的態度問題，而是整個人正在慢慢失去連結感。

所以談復原力，不能只談挫折教育。還要談安全感、談情緒語言、談關係品質、談一個孩子在低潮時能不能被接住。

---

## 9.2 復原力不是天生，更不是靠羞辱長出來

有些孩子確實氣質上比較敏感、有些比較穩定，但這不等於復原力是天生決定好的。

大量研究都指出，復原力非常依賴環境。尤其是孩子生命中是否有穩定、可靠、能幫他調節的大人。

換句話說，一個孩子是否能在受傷後重新站起來，很大程度不是看他夠不夠硬，而是看他跌倒的時候，身邊有沒有一雙手。

說得再白一點，很多時候，連結比糾正更早。孩子不是先被講懂了，才慢慢穩下來；比較常見的是，他先在關係裡被接住，才有能力聽進去那些幫助他長大的話。

這也是為什麼，羞辱式教養常常會養出兩種表面看起來不同、內裡卻很像的結果：

- 一種是非常怕失敗，什麼都不敢試
- 一種是故作不在乎，反正先裝沒差

這兩種都不是復原力。前者是被恐懼綁住，後者是用冷掉來保命。

真正的復原力不需要把孩子逼到牆角才會長。它更像是在一個有支持的環境裡，孩子逐漸學會：

- 失敗不等於完蛋
- 情緒可以被處理
- 我可以請人幫忙
- 我不是只能靠完美才有價值

你會發現，這和前幾章談的創傷、自我批評、羞愧，其實都連在一起。因為一個長期被羞辱的孩子，很難真的有復原力。他比較可能長出的是高警覺、高自責、高表現焦慮。

## 9.3 至少一個穩定的大人，是復原力最重要的底

如果要我只選一個最重要的保護因子，我會選：穩定的大人。

孩子不需要十個完美的大人。他需要的是至少一個，大部分時候可預測、願意靠近、會在關鍵時刻回來的人。

這個人不需要從不失控，也不需要每次都說對話。他只需要讓孩子慢慢知道：

- 我難過時，有人不會嫌我煩
- 我失敗時，有人不會立刻羞辱我
- 我闖禍時，有人會先幫我穩住，再陪我收拾

這樣的經驗會在孩子內心形成一種很重要的底感：世界雖然不總是順利，但我不是孤單的。

**【臨床視角】共同調節（Co-regulation）**

多重迷走神經理論裡有一個關鍵概念：孩子的神經系統在自己的調節能力還沒長成之前，可以「借用」身旁穩定大人的神經狀態來讓自己平靜下來，這個過程叫做「共同調節」（co-regulation）。一個能維持在社交連結狀態的大人——聲調平穩、目光溫和、身體不緊繃——就是孩子神經系統最具體的安全訊號。

換句話說，你不需要完美地執行每一個育兒技巧。但如果你能在孩子崩潰時，讓自己先回到比較穩定的狀態，這件事本身就已經是孩子最需要的。這也是為什麼，父母自己的身心狀態，是孩子復原力最深的底。

**共同調節的三個步驟：**

當孩子情緒崩潰、你感覺自己也快被拉下水時，可以試試這個順序：

一、先回到自己的身體。雙腳踩穩地面，感受腳底的壓力。深吸一口氣，慢慢吐。這不是在忽略孩子，而是在給你的神經系統三秒鐘的時間回到線上。

二、降低你的「訊號強度」。把語速放慢、音量降低、身體蹲下來。孩子的神經系統會讀取你的訊號 —— 如果你的聲音又急又大，他的系統只會更加警戒。反過來，一個穩穩蹲在旁邊、聲音低沉平穩的大人，對孩子來說就是最直接的安全訊號。

三、陪伴，不急著修復。你可以輕輕碰他的肩膀（如果他願意被碰），或者只是安靜坐在旁邊。等他的呼吸開始變慢、身體開始放鬆，你才知道他的前額葉慢慢回來了。這時候再說話，他才真的聽得見。

我曾經遇過一位青少年，學校表現並不好，常常出狀況，家裡關係也緊張。照理說，他是高風險群。可他最後沒有一路往下掉，其中一個關鍵，是他國中時遇到一位導師。那位老師沒有做什麼驚天動地的事，只是穩穩地在。

他遲到，老師會念，但不羞辱。

他考差，老師會談，但不貼標籤。

他情緒來了，老師會先讓他坐一下，再慢慢處理。

多年後他回頭說，當時最重要的不是老師講了什麼大道理，而是他第一次覺得：原來一個大人可以這樣對我。

那種經驗，會成為一個人未來內在穩定的雛形。

我自己在高中時，也曾經很羨慕一位同學和她母親的互動。第一次打電話到她家，是她媽媽接的。她用一種很鬆、很有幽默感的語氣說：「妳找她喔？我剛剛把她沖進馬桶了，我去看看她游回來沒。」我當場愣住，心裡想：原來父母也可以這樣風趣、這樣不讓人緊繃。後來我愈來愈明白，那份幽默不是表面的好笑，而是一種很深的安全感。對帶刺的青少年來說，這種鬆一點、活一點、不急著追著修理的氣氛，本身就是一種修補。它像一個緩衝墊，讓孩子在還沒準備好講真話之前，至少不需要先把盔甲扣得那麼緊。

青春期的孩子常常像在測試一個安全壓桿。他嘴上頂你、推你、冷掉你，表面像在把你推遠，內裡卻常常是在確認：我這麼難搞、這麼刺，你還在不在？所以修補青春期關係時，重要的不只是講對話，還包括你能不能維持一種訊號：我知道你現在很難靠近，但我沒有放棄你。

## 9.4 容許情緒，不會讓孩子更脆弱

很多父母怕孩子太挫折，所以急著安慰、急著解決、急著叫他看開一點。

其實這樣做，常常反而讓孩子更難建立復原力。

因為復原力不是沒有情緒，而是能帶著情緒活下去。

如果一個孩子每次難過都被快速切斷：

- 不要哭
- 這沒什麼
- 想開一點
- 別那麼玻璃心

他會慢慢學到，情緒是麻煩，是不被歡迎的。之後他可能變得不會表達、不敢求助，或者乾脆只用更劇烈的方式讓大人看見。

反過來說，當大人能先接住情緒，孩子反而比較有空間長出調節能力。

例如孩子考差了，與其立刻講分析，不如先說：

「你現在一定很挫折。」

「你這麼難過，表示你其實很在乎。」

「先難過一下，等一下我們再來想怎麼辦。」

這不是寵。這是在幫孩子的大腦從情緒風暴裡慢慢回來。等人回來了，學習和修正才有可能發生。

同樣地，男孩尤其需要這種允許。很多男孩從小就被剝奪了表達脆弱的權利，只剩下生氣或沉默。結果外表看起來很硬，內在其實很怕失敗、很怕丟臉，也很怕自己不夠好。

復原力的培養，需要把「情緒被允許」放進核心，而不是邊角。

但允許情緒，不代表允許所有行為。這個區分很重要。

你可以說：「你現在很生氣，這很正常。」但也可以同時說：「你可以生氣，但不可以打人。」前面是允許感受，後面是設定界線。兩者同時存在，才是完整的安頓。

很多父母擔心：如果我允許孩子的情緒，他會不會越來越失控？答案通常相反。當情緒被接住，孩子反而更容易平靜下來，因為他不需要把音量越調越大才被聽見。真正讓孩子失控的，不是你允許了他的情緒，而是他的情緒從來沒有被真正接住過。

## 9.5 孩子怎麼理解失敗，決定他之後怎麼面對人生

復原力和失敗經驗之間，有很深的關係。

不是孩子失敗越多就越有復原力，也不是完全避免失敗比較好。關鍵在於，他怎麼理解失敗。

如果每一次失敗都被解讀成：

- 你不夠努力
- 你怎麼那麼差
- 你就是不行

那麼孩子很容易把挫敗內化成身份結論。

反過來，如果失敗被放在一個比較完整的脈絡裡：

- 這次沒成功，不代表你不行
- 我們來看是哪裡卡住
- 失敗可以讓人調整方法

孩子就比較可能把挫敗理解成資訊，而不是判決。

很多有憂鬱傾向的孩子，內在其實都有一個非常嚴格的標準。一旦不符合，就整個人崩掉。他們不是不在乎，而是太在乎。太害怕自己失敗之後，就再也不值得喜歡。

這種孩子如果只被要求更積極、更正向，通常幫助不大。更需要的是有人陪他慢慢拆解：一次考不好、一次被拒絕、一次被誤會，真的不等於整個人生完蛋。

這種拆解能力，本身就是復原力的一部分。

## 9.6 培養復原力的四個日常練習

最後，給你幾個很實際的方向。它們不花俏，但會在長期裡累積出力量。

### 一、允許孩子在可承受範圍內受挫

不要把所有坑都替他填平。也不要把每一次失敗都當成你教養失敗的證據。

孩子需要在有人在旁邊的情況下，經驗一點點不順、等待、輸掉、做不好。這些經驗如果被接住，會變成能力。

很多父母會想：孩子現在這麼小，就算陪他走過這些，他長大後也未必記得。可大腦不是只靠那些說得出來的回憶在長大。很多真正影響一個人的，是內隱的經驗。孩子也許不記得某次輸了比賽後，你陪他走回家的那段對話；也許不記得某次旅行、某次練習、某次跌倒後你蹲下來抱他的表情。但他的神經系統會記得：我失敗過，但沒有被丟下。我很挫折，但有人陪我一起走過去。這些經驗，會慢慢長成他日後面對人生時的底氣。

### 二、情緒先於教訓

每次孩子出事時，先問自己：

他現在最需要的是修理，還是穩住？

順序對了，教導才進得去。

### 三、把求助正常化

讓孩子知道，遇到困難可以找人，不代表沒用。

你也可以直接示範：

「媽媽今天太累了，我需要你爸爸幫忙。」

「這件事我不會，我們一起問別人。」

求助本身就是復原力，不是例外。

### 四、練習修補，而不是追求永不失誤

沒有哪個家庭能完全不受傷。真正重要的是，受傷之後有沒有回來。

你可以對孩子說：

「剛剛我口氣很差，對不起。」

「你剛剛很挫折，我沒有先懂你，我們重來一次。」

這種修補經驗，會讓孩子知道：關係不是一壞就完了。壞了可以修。這本身就是很深的復原力教育。

## 9.7 情緒詞彙：能說出來，就比較不會被吞掉
能夠用語言描述自己的感受，是復原力的基礎。

研究顯示，能夠精確命名情緒的孩子（「我不只是難過，我感到被背叛」），比只能說「不好受」的孩子，更容易從創傷中恢復。神經科學家 Lisa Feldman Barrett 的研究指出，情緒詞彙越豐富的人，大腦的前額葉皮層越能有效地調節杏仁核的活動。白話一點說：能說出「我現在什麼感覺」的孩子，大腦比較不容易被情緒劫持。

怎麼幫孩子建立情緒詞彙？很簡單：幫他命名。

當他哭的時候，你不是說「不要哭」，而是說「你現在是不是覺得很委屈？因為同學沒有邀請你。」

當他生氣的時候，你不是說「生氣也沒用」，而是說「你很生氣對不對？因為你覺得不公平。」

每一個情緒被命名的瞬間，都是大腦前額葉在跟杏仁核說話的過程。練得越多，孩子就越能自我調節。

** 不同年齡層的命名示範：**

幼兒（2-5 歲）只需要簡單的詞彙：「你很開心」「你很生氣」「你好像有點害怕」。這個階段的重點不是精確，而是讓孩子知道「感受有名字」。

學齡期（6-12 歲）可以開始引入更細緻的詞彙：「你覺得這件事不公平，所以很氣憤」「你跟好朋友吵架了，是不是既難過又懊惱？」你可以用「是不是覺得……」的句型，幫孩子猜，而不是直接告訴他。

青少年（13 歲以上）可以嘗試更複雜的情緒語言：「你現在是不是有點矛盾？一方面想獨立，一方面又覺得需要幫忙」「那件事讓你覺得被誤解了，對吧？」青少年最怕被「教育」，你可以多用「我觀察到」「我猜」的開頭，留空間讓他自己接話。

** 給讀者的情緒詞彙參考表：**

| 基本情緒 | 進階情緒 | 更精細的描述 |
|---------|---------|------------|
| 難過 | 委屈、遺憾、失落 | 被忽略的寂寞、沒被看見的空虛 |
| 生氣 | 氣憤、煩躁、不甘 | 被誤解的委屈、努力被否定的挫敗 |
| 害怕 | 焦慮、緊張、不安 | 對未知的恐懼、怕讓人失望的壓力 |
| 開心 | 滿足、感恩、驕傲 | 被認可的踏實、努力的成果被看到的喜悅 |
| 尷尬 | 羞愧、丟臉、難為情 | 被當眾指出錯誤的無地自容 |

這份表不是讓你拿來考孩子的，而是給你自己參考 —— 當你不知道怎麼幫孩子命名時，可以從中找到適合的詞。

## 9.8 掌控感的建立
創傷的核心體驗之一是「無力感」—— 我對發生在我身上的事情沒有控制權。

培養復原力的方法是：在安全的範圍內，給孩子越來越多的選擇權和掌控權。

三歲的小宇早上不肯穿外套，媽媽說：「今天冷，不穿外套會著涼，一定要穿。」小宇躺在地上尖叫。第二天媽媽換了策略：「今天有兩件外套，你要紅色的還是藍色的？」小宇想了想，說「藍色的」，然後自己穿上了。

這不是什麼高深的技術，只是一個小小的選擇權。但對三歲的孩子來說，「我自己決定的」跟「被迫的」是完全不同的體驗。

小學二年級的美美不喜歡上鋼琴課，每次去之前都哭。媽媽跟她聊了很久，發現她不是不喜歡音樂，是每次都被規定要練指定的曲子，讓她覺得「這不是我的」。後來媽媽跟老師商量，每週讓美美自己選一首她想彈的歌。幾個月後，美美開始主動練琴了。

十四歲的阿凱想參加學校的籃球隊，但練習時間跟補習衝突。爸爸沒有直接替他決定，而是跟他一起列出「參加的好處」「不參加的好處」「可能遇到的困難」，讓他自己做決定。阿凱最後選擇了參加籃球隊，他說：「雖然可能會累，但這是我的選擇。」

每一個「我自己決定」的瞬間，都在告訴孩子：你有能力影響你的生活。這種感覺累積起來，就是面對逆境時「我不會完全無助」的底氣。

## 9.9 失敗的日常化
如果一個孩子從來沒有失敗過，他怎麼學會應對失敗？

讓孩子在安全的環境裡體驗失敗。比賽沒得名次、考試考砸了、跟朋友吵架了、被拒絕了。這些都是練習。

你的角色不是幫他避免失敗，而是在他失敗之後陪他走過去。問他：「發生了什麼事？你現在感覺怎麼樣？你覺得下次可以怎麼做？」

讓他從你的態度裡學到：失敗不是世界末日。它只是生活的一部分。

但說起來容易，做起來很難。很多父母在孩子失敗的時候，自己的情緒比孩子還激動 —— 比賽輸了，爸爸比孩子還生氣；考試考差了，媽媽比孩子還焦慮。這些反應背後，往往是我們自己小時候的經驗在作祟。我們那個年代，「失敗」代表被罵、被嘲笑、被認為「不夠好」。所以我們長大後，看到孩子失敗，身體裡那個小時候被罵的自己就醒了過來。

如果你發現自己有這個模式，可以先做一件事：在孩子失敗的時候，先安撫自己。深呼吸，告訴自己：「這是他的經驗，不是我的。他需要的是一個陪他的人，不是另一個壓力源。」

而如果你養的是男孩，這件事尤其值得留意。PISA（國際學生能力評量計畫）的研究數據顯示，在「對失敗的恐懼」這個指標上，男孩的平均得分顯著高於女孩 —— 也就是說，男孩比女孩更害怕失敗。這個現象在東亞國家尤其明顯。但問題是，男孩不被允許說出來。當一個女孩說「我好怕考不好」，周圍的人多半會安慰她；但當一個男孩表現出同樣的焦慮，他更可能聽到的是「男孩子哭什麼」或「有什麼好怕的」。於是他學會了一件事：怕失敗可以，但不能被發現。他會用沉默、生氣、甚至故意表現不在乎來掩蓋內心的恐懼。

所以，如果你養的是男孩，在失敗的日常化這件事上，你需要多走一步：不只是讓他失敗、陪他走過去，還要主動幫他把那份「不敢說的害怕」說出來。「你現在是不是覺得有點丟臉？」「考試前會不會有點緊張？」—— 這些問題，對很多男孩來說，是第一次有人允許他們這樣感覺。

心理學有一個概念叫「創傷後成長」（Post-Traumatic Growth, PTG）。研究者 Tedeschi 和 Calhoun 發現，有大約 50% 到 70% 經歷過重大創傷的人，能恢復到創傷前的狀態，有些人甚至在某些面向上變得「比以前更好」—— 他們對人際關係的珍惜更深了，對生命的意義感更強了，對自己的能力也更有信心了。

這不是說創傷是好事。創傷很痛，沒有人應該被創傷。但人類的身心系統有一種驚人的修復能力。關鍵在於：這個人是不是有一個安全的環境去消化這段經歷、是不是有人陪他一起找到意義、是不是被允許慢慢來。

對孩子來說，日常的小失敗就是創傷後成長的「預防針」。每一次他從失敗中站起來，都在告訴自己：「我犯錯了，但我沒有碎掉。」

## 9.10 你自己的復原力，就是孩子最早的教科書
孩子最終會從你身上學到的，不只是怎麼面對成績、挫折和失敗，也包括：

一個人跌倒時，怎麼對待自己。

你若每次出錯都羞辱自己，孩子會學到：失敗很可怕。

你若每次壓力一來就假裝沒事，孩子會學到：脆弱不能說。

但如果你能在失手時承認、在難過時不否認、在撐不住時願意求助，孩子會看到另一種活法：

原來人可以不完美，卻仍然有力量。

這份力量，不喧嘩，也不戲劇化。它很普通。可能只是你在某個疲倦的晚上，對孩子和自己都說了一句：

「今天真的很糟，但我們明天再試一次。」

很多時候，復原力就是這樣長出來的。不是因為你從來沒倒下，而是因為在一次又一次倒下之後，你學會了怎麼把自己慢慢扶回來。

最後，我想提一個常被忽略的事實：孩子的復原力，很大程度上是在模仿你的。

如果你在孩子面前摔倒之後能夠坦然地說「哎呀摔倒了，好痛，沒關係再試一次」，你的孩子就會學到：摔倒不是末日。

如果你在孩子面前遇到挫折時總是暴怒、自責、崩潰，你的孩子就會學到：失敗是可怕的。

你不是要當一個「不會失敗」的父母。你是要在失敗的時候示範：怎麼接受它、怎麼處理它、怎麼走過去。

我想分享自己的經歷。

幾年前，我花了好幾個月準備一場大型專業演講。從題目設計、文獻整理、投影片製作到講稿撰寫，前前後後投入了至少一百個小時。演講當天，我興奮又緊張地站在台上，開場沒多久，投影設備突然故障，螢幕一片黑。台下幾百雙眼睛看著我。

那個瞬間，我的大腦一片空白。心跳加速，手心冒汗，臉部發燙。我可以感覺到恐慌像潮水一樣湧上來。

我深呼吸了一下，對著觀眾說：「看來投影有點想法，那我們就先用說的吧。」台下有人笑了。我放下講稿，用最自然的方式把內容講完。結束的時候，有好幾位參與者走過來說：「妳處理得好好，一點都看不出來妳緊張了。」

回家那天晚上，孩子看到我，問：「媽媽今天好玩嗎？」我蹲下來跟她說：「今天發生了一點小狀況，投影機壞了，媽媽本來有點慌張，但後來還是講完了。」

女兒歪著頭想了想，說：「所以有時候事情會不聽話，但我們還是可以做完？」

「對，就是這樣。」

那一次，我做的其實就是本書前面提到的幾件事：我沒有壓抑自己的緊張（情緒詞彙 —— 我說出了「有點慌張」），我承認了自己的脆弱，然後我帶著脆弱繼續完成了該做的事。

我不知道這段對話在她心裡留下了什麼。但我知道，那天晚上我在做的，就是在示範「怎麼在事情不順利的時候，不讓自己崩潰，而是帶著慌張繼續往前走」。

還有一次，我不小心把女兒手做了一整個下午的手工撕壞了。她當場大哭。我的第一反應是自責 ——「你怎麼這麼粗心」「明明是她最珍惜的東西」。那個自我批評的聲音差點就要衝出來。但我忍住了，蹲下來看著她，說：「媽媽真的很抱歉，是媽媽不小心弄壞的。媽媽知道妳很傷心，因為那是妳很用心做的。」女兒哭了很久，後來她說：「媽媽，你弄壞了但是你沒有騙我說是它自己掉的。」

我不知道她哪來的這種觀察力。但那句話提醒了我：孩子在看的不只是你做了什麼，還有你怎麼面對你做錯的事。道歉、承認錯誤、承擔後果 —— 這些都是復原力的具體表現。當父母願意示範這些，孩子就會學到：犯錯不可怕，可怕的是不承認、不面對。

「你的孩子不記得你對他說過的每一句道理，但他會記得你面對困難時的樣子。」

## 9.11 兒童復原力量表：你的孩子夠「有彈性」嗎？
以下是一個簡單的自我評估工具，幫你觀察孩子的復原力狀態。這不是診斷工具，只是一個讓你更有方向去了解孩子的參考。

請回想孩子最近六個月來的表現，在每個項目上勾選「經常」「有時」或「很少」。

| 項目 | 經常 | 有時 | 很少 |
|------|------|------|------|
| 1. 遇到困難時，願意嘗試不同的解決方式 | | | |
| 2. 失敗或被拒絕後，能在合理的時間內恢復情緒 | | | |
| 3. 能用語言表達自己的感受（不只是哭或發脾氣） | | | |
| 4. 願意尋求幫助（向父母、老師、朋友） | | | |
| 5. 面對新環境或新挑戰時，雖然緊張但願意嘗試 | | | |
| 6. 能從失敗的經驗中學到東西（「下次我會……」） | | | |
| 7. 有一個他信任、願意分享心事的人 | | | |
| 8. 在日常生活中有自己能做主的範圍（選衣服、選活動等） | | | |
| 9. 遇到不順利的事，不會立刻認定「我就是不行」 | | | |
| 10. 跟朋友吵架後，有修復關係的意願和行動 | | | |

** 怎麼看結果：**

「經常」6 個以上：孩子的復原力發展得不錯。繼續保持你正在做的事。

「經常」3-5 個：有基礎，但某些面向可以加強。可以對照前面的四個支柱，看看哪一塊比較弱。

「經常」2 個以下：孩子可能需要更多的支持。這不代表他「有問題」，而是他還在等一個更安全的環境、一個更穩定的陪伴者。如果孩子在情緒上長期處於困擾狀態，建議尋求專業心理師的協助。

請記住：這份量表觀察的是「目前的狀態」，不是「永遠的標籤」。孩子的復原力會隨著環境和經驗而改變。而你能做的，就是提供那個讓他改變的環境。

## 9.12 一個從挫折中站起來的故事
我想用一個完整的故事，把這一章的內容串起來。

小傑是個五年級的男孩，他從小就喜歡畫畫。三年級的時候，美術老師在全班面前表揚他的作品，他高興了好幾天。從那時候開始，他一直覺得自己是班上「畫得最好的人」。

五年級的學校美術比賽，他信心滿滿地報了名。準備了兩個星期，每天放學後都留在教室畫。比賽那天，他帶著自己最滿意的作品去交件。結果公布，他沒有得獎。連優選都沒有。

回家的路上，他一句話都沒說。媽媽從他的表情裡看出來了，問他：「比賽結果怎麼樣？」

「沒得名次。」他的聲音很輕。

媽媽忍住「沒關係啦下次再努力」的衝動，說了一句：「那一定很難受吧。」

小傑突然就哭了。他說：「我準備了那麼久，結果什麼都沒有。我一定很爛。」

媽媽沒有急著安慰他，而是坐在他旁邊，安靜地等他哭完。然後問他：「你願意跟媽媽說說，最讓你難過的是什麼嗎？」

「就是…… 我以為我畫得很好，結果別人畫得比我好太多。我以前覺得自己很厲害，現在我覺得自己什麼都不是。」

「嗯，你不是『什麼都不是』。你是發現了，原來有比自己更厲害的人。這個感覺的確很衝擊。」

那天晚上，媽媽沒有多說什麼。她只是做了一件事 —— 把小傑以前畫的畫，從冰箱上、從房間牆上，一張一張取下來，跟他一起看。「你看，這幅是小班時候畫的，這幅是一年級的，這幅是去年的。你看，你的畫一直在進步，對不對？」

小傑看著那些畫，表情慢慢從沮喪變成若有所思。「好像…… 是有變好一點。」

「比賽沒得名次，不代表你沒有進步。它只代表，在那一天、那些評審的眼裡，有別人的作品更符合標準。但你的畫，對媽媽來說，一直都是最好的。」

幾天後，小傑跟同學聊天時提起了這件事。同學說：「我上次參加作文比賽也沒得名次，我後來才知道，全班只有五個人報名，最後得獎的都是六年級的。」小傑笑了：「那我也沒輸給五年級的啊。」他有點驚訝地發現，原來「沒得名次」不代表「我很差」，也可能只是「評審的標準跟我不一樣」。

媽媽後來跟我分享這件事時說了一個讓我很感動的細節。那天晚上，小傑睡覺前突然跟媽媽說：「謝謝你沒有說『沒關係啦』。」媽媽愣了一下，問：「為什麼？」

「因為『沒關係』就是說這件事很嚴重，妳在安慰我。但我當時不是想被安慰，我只是想有人知道我很難過。」

一個十一歲的孩子，能說出這樣的話。這就是情緒詞彙和穩定關係一起作用的結果。他不但能命名自己的感受，還能分辨「被安慰」和「被理解」的不同。

第二天，小傑主動拿出畫簿開始畫新的東西。媽媽偷偷瞄了一眼，畫的是一個站在頒獎台旁邊的小人，手裡拿著自己的畫，嘴角有點往下，但背挺得很直。旁邊寫了一行字：「下次，我會再試一次。」

## 本章小結

復原力是可培養的技能，不是天生的性格特質。四個支柱是：穩定的成人關係、情緒詞彙、掌控感、失敗的日常化。

PISA 研究顯示男孩比女孩更害怕失敗 —— 但男孩不被允許說出來。幫孩子命名情緒，就是在訓練他的大腦自我調節。

孩子的復原力在模仿父母的應對方式。你不需要是完美父母，你只需要在跌倒時示範怎麼站起來。用兒童復原力量表，觀察孩子目前的狀態，找到可以加強的方向。先有連結，孩子才比較長得出面對挫折的力量。

第三篇走到這裡，你已經回答了這本書最核心的問題：育兒不只是養孩子，也是重新養育自己。你學到了重新養自己的具體入口，也理解了復原力不是硬撐，而是在痛裡依然能維持連結。

最後一篇，我們要帶著這些覺察，回到最真實的教養現場。這次，你的眼光已經不一樣了。
