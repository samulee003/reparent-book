## 一、總編輯審稿（全書架構與商業定位）

### 1. 整體評價

方案 C 的「雙鏡面型」結構是三套方案中最大膽的重排。核心邏輯是「先穩住現在的自己，再回頭看傷的來源」，這與主流心理治療的第一原則（先建立安全感）高度一致。自我慈悲提前到第三章，讓讀者在翻開書的前三分之一就獲得實質的安撫工具，是最友善讀者的設計。

### 2. 架構問題

#### P0：篇章不均衡

| 篇 | 章數 | 中文字數 | 佔比 |
|---|---|---|---|
| 第一篇 先停下來，看見自己 | 3 章 | 14,878 | 18.2% |
| 第二篇 舊傷從哪裡來 | 4 章 | 29,182 | 35.7% |
| 第三篇 重新養育自己 | 2 章 | 15,845 | 19.4% |
| 第四篇 帶著覺察陪孩子長大 | 3 章 | 21,293 | 26.1% |

**問題**：第二篇佔全書 35.7%，是第三篇的近兩倍。第三篇只有兩章，作為「全書最核心」的篇章顯得過於單薄。

**建議**：
- 方案一（推薦）：將 ch09 復原力移至第四篇開頭，改為 3-4-1-4 結構，但第三篇僅剩一章會更薄。不建議。
- 方案二（推薦）：為第三篇增寫一章「覺察式教養：從知道到做到」，作為 Part 2 到 Part 4 之間的銜接橋樑。但增章風險較高。
- 方案三（最穩）：維持現狀 3-4-2-3 結構，但在前言和目錄中強調第三篇是「全書高潮段」，用篇章頁設計（加大留白、加引言）來補償視覺上的不對稱。

#### P0：ch02 內在批評章字數過短

ch02（3,356 字）是全書最短章節，不到 ch04（10,040 字）的三分之一。放在第一篇第二章的位置，承擔「認出內在批評者」的核心任務，篇幅明顯不足。

**建議**：擴充至 5,000-6,000 字。可從以下方向補充：
- 增加一個完整案例（目前只有一個母親案例，可再加一個父親案例或不同背景的案例）
- 增加「內在批評者 vs. 健康的自我反省」的區分段落
- 加入一個「寫信給內在批評者」的練習（這也能修復 ch08 的斷裂引用）

### 3. 書名與結構的契合度

書名《當了父母，才開始重新養育自己》對應方案 C 的結構非常精準：Part 1-2 是「當了父母」觸發覺察的歷程，Part 3 是「重新養育自己」的核心，Part 4 是養完自己後回到教養。建議副標題可以考慮調整為「心理師的覺察式育兒練習」→「心理師的雙鏡面育兒書」以呼應結構，但此為行銷層面，可後議。

### 4. 市場定位

方案 C 的「先療後解」邏輯，與市面上大多數育兒書（先講問題再講方法）不同，具備差異化優勢。但需在行銷文案中清楚傳達「這不是一本教你怎麼搞定孩子的書，而是先搞定你自己」。

## 五、臨床心理學家審稿（理論框架與臨床準確性）

### 1. 理論框架整合

方案 C 的核心理論框架：
- **IFS（內在家庭系統）**：主要出現在 ch02（臨床視角）、ch08（重新養自己）
- **多重迷走神經理論**：主要出現在 ch02（臨床視角）、ch04、ch06
- **基模治療**：主要出現在 ch05（CEN 與早期適應不良基模）

**評價**：三大理論在方案 C 中的分佈合理。「先穩定再回溯」的順序與臨床實務中的 Phase-based Treatment（階段式治療）一致：Phase 1 安全感建立（Part 1）→ Phase 2 創傷處理（Part 2）→ Phase 3 整合（Part 3-4）。

### 2. 方案 C 獨特的臨床優勢

#### 先建立安全感再看傷

這是方案 C 與其他方案最大的區別。在臨床實務中，我們不會一開始就帶個案回到創傷現場，而是先確保個案有足夠的情緒調節能力和自我安撫資源。

方案 C 正是這樣做的：
- ch01：看見你的處境（正常化）
- ch02：認出內在批評者（命名與外化）
- ch03：學習自我慈悲（建立安全基地）
- ch04-07：才開始回溯童年創傷

這個順序在臨床上是最安全的。

#### P1：ch02 內在批評出現在 ACEs 之前的解釋不足

**問題**：讀者可能會問「這個內在批評的聲音到底從哪裡來的」— 因為 ACEs 和 CEN（解釋來源的章節）要到 ch04-05 才出現。

**目前處理**：ch02 開頭（第 3 行）有一句「上一章，我們看見了新手父母的處境…」，但沒有預告「這個聲音的來源我們後面會詳談」。

**建議**：在 ch02 中增加一段鋪墊，例如：「這個聲音到底是從哪裡來的？我們會在第二篇好好回答這個問題。但在那之前，我想先做一件事：不管它從哪裡來，先幫你學會在它出現的時候，不再被它吞掉。」

## 八、總結

方案 C「雙鏡面型」是三套方案中**讀者體驗最友善、臨床邏輯最正確**的結構。「先穩定自己，再回頭看傷」的順序，不僅符合心理治療的階段式治療原則，也讓讀者在第三章就獲得自我慈悲的實質安撫工具。

主要的挑戰在於：
1. **ch02 篇幅不足**（最短章放在核心位置）
2. **ch03 與 ch08 的內容重疊**（自我慈悲主題跨兩篇出現）
3. **5 個章間轉場缺失**（重排後的銜接未完全重寫）
4. **結構元素不統一**（本章小結、臨床視角分佈不一）

以上問題均可修復，且修復後方案 C 將是最完整、最具出版品質的版本。

**審稿團隊共識**：建議採用方案 C，並按 P0→P1→P2 優先級執行修改。
